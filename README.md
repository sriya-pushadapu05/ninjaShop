# NinjaShop Automation Framework

## Project Description

This project is a Selenium Java Automation Framework developed using Page Object Model (POM) for an e-commerce demo application.

---

## Features

* User Registration, Login, and Logout Automation  
* Product Search and Navigation  
* Shopping Cart Operations (Add, Update, Remove)  
* Cart Count Validation (increase/decrease)  
* Checkout Flow Automation  
* Form Validations (empty fields, invalid inputs)  
* Screenshot capture on test failure  
* ExtentReports for HTML reporting  

---

## Tech Stack

* Java  
* Selenium WebDriver  
* TestNG  
* Maven  
* WebDriverManager  
* ExtentReports  

---

## Reports

Execution reports are generated in the `/reports` folder using ExtentReports.

---

## Screenshots

Screenshots are captured automatically on test failure and stored in `/screenshots`.
